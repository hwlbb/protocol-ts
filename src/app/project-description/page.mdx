# 项目说明

使用TypeScript，Next.js，MDX，Tailwind CSS，Headless UI构建的现代化API文档网站。

## 项目结构

本项目采用了典型的Next.js应用结构,主要包含以下几个部分:

### `/src/app`

这个目录包含了所有的页面和路由。在Next.js 14中,这个目录使用了新的App Router。主要文件包括:

- `layout.tsx`: 定义了整个应用的布局结构
- `page.mdx`: 首页内容
- 其他以文件夹形式组织的子页面,如`quickstart`, `sdks`等

### `/src/components`

这个目录包含了所有可重用的React组件。主要组件包括:

- `Navigation.tsx`: 定义了网站的导航结构
- `Header.tsx`: 网站的头部组件
- `Footer.tsx`: 网站的底部组件
- `Layout.tsx`: 定义了页面的整体布局
- 其他功能性组件,如`Button.tsx`, `CodeGroup.tsx`等

### `/src/styles`

包含全局样式文件,主要是`tailwind.css`。

### `/public`

存放静态资源,如图片、字体等。

### 根目录文件

- `next.config.mjs`: Next.js的配置文件
- `tailwind.config.js`: Tailwind CSS的配置文件
- `tsconfig.json`: TypeScript的配置文件
- `package.json`: 项目依赖和脚本定义
